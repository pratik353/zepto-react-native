# OPEN APP ON MOBILE --> WIRELESS
1. Install react native dev app on mobile
2. open app --> you will see error with red screen 
3. shake the mobile --> will open dev menu list 
4. click on settings 
5. click on Debug server host & port for device
6. add your wifi ip --> 192.***.**:8081 ( 8081 --> APP PORT )
7. then reload the application from terminal
